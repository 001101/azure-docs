# Manage Azure resources by creating automation tasks (preview)

To help you manage Azure resources more easily, you can create and automate tasks for a specific resource by using automation task templates, which vary based on the resource type. For example, for a virtual machine resource, you can set up an automated task that sends you the monthly cost for that virtual machine or that turns on and turns off that virtual machine.

While you create automation tasks from the user experience for a specific Azure resource, these tasks are actually powered and run by [Azure Logic Apps](../logic-apps/logic-apps-overview.md). This underlying platform makes possible for you to examine the run history for a task and also the capability to edit or customize the task to your needs.
