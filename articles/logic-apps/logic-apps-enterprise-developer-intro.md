# Develop custom solutions for your organization with Azure Logic Apps

As an enterprise developer, you can empower your organization 
to build robust tailored solutions with Azure Logic Apps. 
You can create and use custom APIs for connecting to your data and web services, 
embed logic apps in your organization's websites, 
and work with data in the Common Data Service.

## Build custom API connectors

To connect to your organization's data 
and web services with Logic Apps, 
[learn how you can develop custom connectors](register-custom-api.md)

## Build code with Azure Functions

To extend your logic apps with custom server-side code, 
[learn how you can write and use Azure functions](**NEED TOPIC LINK**}.

## Embed your logic apps

To create integrated solutions and surface workflows or processes 
where people in your organization already do their work, 
[learn how you can embed logic apps directly into your website experiences](**NEED TOPIC LINK**)
