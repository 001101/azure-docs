# Azure IoT Edge Device Security Certification

Azure IoT Edge offers a comprehensive device certification program among which is certiification for security.  Security certification for IoT Edge devices primarily exists as a tool of communications to inform on investments made towards securing IoT devices.  It helps to showcase investments by device makers and guide informed choices by device buyers.
