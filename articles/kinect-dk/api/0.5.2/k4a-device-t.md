# k4a_device_t handle

Handle to a k4a device. 

## Syntax

```C
typedef struct { } k4a_device_t;
```
Handles are created with 
[k4a_device_open()](~/api/0.5.2/k4a-device-open.md)
 and closed with 
[k4a_device_close()](~/api/0.5.2/k4a-device-close.md)
. Invalid handles are set to 0.

## Requirements

Requirement | Value
------------|--------------------------------
 Header | k4atypes.h (include k4a/k4a.h)
