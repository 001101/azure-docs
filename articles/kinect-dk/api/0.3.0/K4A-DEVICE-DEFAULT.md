# K4A_DEVICE_DEFAULT definition

Indicates the caller of 
[k4a_device_open](~/api/0.3.0/k4a-device-open.md)
 wants to open the default sensor. 

## Syntax

```C
#define K4A_DEVICE_DEFAULT (0)
```

## Requirements

Requirement | Value
------------|--------------------------------
 Header | k4atypes.h (include k4a/k4a.h)
