# K4A_PLUGIN_DYNAMIC_LIBRARY_NAME definition

Expected name of plugin's dynamic library. 

## Syntax

```C
#define K4A_PLUGIN_DYNAMIC_LIBRARY_NAME "depthengine"
```

## Remarks
When the K4A SDK tried to load the depth engine, it will attempt to load a dynamic library named "depthengine"
