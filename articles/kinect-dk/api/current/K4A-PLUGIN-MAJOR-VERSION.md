# K4A_PLUGIN_MAJOR_VERSION definition

Current Version of the k4a Depth Engine Plugin API. 

## Syntax

```C
#define K4A_PLUGIN_MAJOR_VERSION 1
```

## Remarks
The MAJOR version must be updated to denote any breaking change that would cause an older SDK to not be able to use this plugin.K4A plugin major version
