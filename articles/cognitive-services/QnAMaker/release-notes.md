# QnA Maker release notes

Learn about the last five service releases here. Older release notes are kept in the release notes archive.  



## Version 4.2.0

Version update status is now displayed on the **[User Settings](https://www.qnamaker.ai/UserSettings)** page. Any knowledge bases that should be updated have a link next to the version number. All knowledge bases in a subscription that need to be updated, will be updated when you update any one of the knowledge bases.
