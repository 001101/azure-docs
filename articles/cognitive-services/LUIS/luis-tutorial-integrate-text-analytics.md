# Get sentiment analysis from LUIS endpoint queries
You can integrate your LUIS app with [Text Analytics](https://docs.microsoft.com/azure/cognitive-services/text-analytics/) to get sentiment analysis for user utterances.

## 

## Sentiment analysis provides emotion scores
Sentiment analysis provides positive, negative and neutral scores. Your app or chat bot uses this score to enrich the conversation experience for the user.
