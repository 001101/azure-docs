To use the latest version of the Text Analytics client library v3, you will need to download the latest software package in the `Azure.AI.TextAnalytics` namespace. The **Setting up** section in the [quickstart article](quickstarts/text-analytics-sdk.md) lists the commands you can use for your preferred language, with example code.

The v3 Text Analytics client library is available in the following programming languages, as part of the Azure SDK: C#, Python, Java, JavaScript
