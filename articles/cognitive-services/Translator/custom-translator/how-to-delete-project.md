# Delete a project

1.  Hover on any project record and click on the trash icon.

    ![Delete project](media/how-to/ct-how-to-delete-project.png)

2. Confirm deletion. Deleting a project will delete all models that were created
within that project. 


    ![Delete confirmation dialog](media/how-to/ct-how-to-delete-project-confirm.png).

>Note: Deleteing project will not affect your  documents.
