# Scenario Availability

The Speech service SDK features many scenarios across a wide variety of programming languages and environments.  Not all scenarios are currently available in all programming languages or all environments yet.

- Speech-recognition, Intent, Translation: Available everywhere there is an an arrow link <img src="media/index/link.jpg" height="15" width="15"></img> in the quickstart-table [here](https://aka.ms/csspeech).
- Text-to-speech: Available for C++/Linux, C++/Windows, C#/Windows.  REST can be used in every other situation.
- Voice-first virtual assistant: Available for C#/UWP
- Keyword spotter: Available for C++/Windows&Linux, C#/Windows&Linux, Python/Windows&Linux, Java/Windows&Linux&AndroidSpeechDeviceSDK
- Codec compressed audio input: Available for C++/Linux, C#/Linux, and Java/Linux
