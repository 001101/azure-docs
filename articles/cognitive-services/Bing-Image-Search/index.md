    <h1>Bing Image Search API Documentation</h1>
    <div class="intro" style="min-width: 200px">
        <p>Bing Image Search API provides an experience similar to Bing.com/Images by returning image results that Bing determines are relevant to the user's query. The results also include four different types of search suggestions. Learn how to search images in different ways with our quickstarts, tutorials, and how-to guides.</p>
