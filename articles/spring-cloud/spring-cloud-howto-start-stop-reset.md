# How to start, stop, and restart your Azure Spring Cloud application

Once you have an application deployed, you can **Start**, **Stop**, and **Restart** it via the Azure portal.

1. Go to your Azure Spring Cloud service instance in the Azure portal
1. Click on the **Application Dashboard** tab.
1. Click on the application you want to change the state of.
1. In the **Overview** page for that application, there is a **Start/Stop** button and a **Restart** button at the top of the page. Click the buttons to change the application's state.
