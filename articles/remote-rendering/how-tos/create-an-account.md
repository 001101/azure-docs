# Create an account for Azure remote rendering

The following steps are involved to create an account for the Azure remote rendering service:

1. Go to https://aka.ms/MixedRealityPrivatePreview
2. Click on 'All services'
3. In the search box, type 'Remote Rendering'
4. Select 'Remote Rendering Accounts'
5. Click on  '+Add'
    1. Set 'Resource Name' to the name of the account
    2. Update 'Subscription' if needed
    3. Set 'Resource group' to a resource group of your choice
    4. Set 'Location' to 'East US2'
6. Once the account is created, navigate to it and:
    1. In the Overview tab, note the 'Account ID'
    2. In the Keys tab, note the 'Primary key' - this is the account's secret account key
