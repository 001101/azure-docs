# Sample model

A sample model is provided that can be used to test the Azure Remote Rendering service. This model has been converted to the proprietary binary format (*.arrAsset*) so it is ready to be used via its SAS URI.

For best loading performance, pick the SAS URI from the region that is used to initialize the service.
