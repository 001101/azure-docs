# Azure Data Factory Alter Row Transformations

[!INCLUDE [notes](../../includes/data-factory-data-flow-preview.md)]

![Alter row settings](media/data-flow/alterrow1.png "Alter Row Settings")

Use the Alter Row transformation to set insert, delete, update, and upsert policies on rows. You can add one-to-many conditions as expressions. Each of those conditions can result in a row (or rows) being inserted, updated, deleted, or upsert.
