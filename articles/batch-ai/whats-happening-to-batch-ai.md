## How does this affect me?

Beginning December 4, 2018, we will no longer make updates to Batch AI. After this date, you will no longer be able to create new subscriptions. Existing subscriptions will continue to run through March 2019.

By the end of March 2019, we expect to have full feature parity between Batch AI and Azure Machine Learning.
