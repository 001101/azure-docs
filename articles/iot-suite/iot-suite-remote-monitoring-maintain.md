# Toubleshoot and remediate device issues to maintain solution health

This how-to guide shows you how to use the dashboard to troubleshoot and remediate device issues. As an operator, you use thes capabilities to maintain the solution health.

In this how-to guide, you learn how to:

>[!div class="checklist"]
> * View the details of an alarm
> * Acknowledge an alarm
> * View the device that triggered the alarm
> * Schedule a task
> * Edit the rules

## Prerequisites

To follow this how-to guide, you need a deployed instance of the remote monitoring solution in your Azure subscription.

If you haven't deployed the remote monitoring solution yet, you should complete the [Deploy the remote monitoring preconfigured solution](iot-suite-remote-monitoring-deploy.md) tutorial.
