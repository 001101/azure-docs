# Bulk import data into a dataset

This guide will show you how to use the [Dataset Import API]() to bulk import data into a dataset from data that was uploaded to the Azure Maps Data service.

## Prerequisites

You need to obtain a primary subscription key by making an Azure Maps account. The primary subscription key is a required parameter for all Azure Maps APIs.

In this article, the Postman application is used to call the Azure Maps APIs. But you may use any API development environment.

Before you import data to the Azure Maps resources, make sure that your DWG package adheres to the [DWG package requirements]().
