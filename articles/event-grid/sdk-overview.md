# Event Grid SDKs for management and publishing

Event Grid provides SDKs that enable you to programmatically manage your resources and post events.

## Management SDKs

The management SDKs enable you to create event grid topics and subscriptions. Currently, the following SDKs are available:

* .NET
* Go
* Java
* Node
* Python
* Ruby

## Publish SDKs

The publish SDKs enable you to post events to topics. Currently, the following SDKs are available:

* .NET
* Go
* Java
* Node
* Python
* Ruby
