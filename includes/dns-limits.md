**Public DNS zones**

| Resource | Default limit |
| --- | --- |
| Zones per subscription |100 <sup>1</sup> |
| Record sets per zone |5,000 <sup>1</sup> |
| Records per record set |20 |

<sup>1</sup>If you need to increase these limits, contact Azure Support.
